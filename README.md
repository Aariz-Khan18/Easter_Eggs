# 🚀 Galaxy Defender

A fast-paced, frontend-only Space Shooter game built entirely from scratch without any external libraries or game engines. Survive the alien waves and defeat the Level 10 Boss!

## 🎮 Features

● **Auto-Fire Mechanics:** Your ship fires automatically—focus entirely on dodging and positioning.
● **Progressive Difficulty:** 10 levels of escalating speed and complex enemy movement patterns (zig-zag tracking).
● **Boss Battle:** Face off against a high-HP Boss ship at Level 10.
● **Power-up Drops:** - `[U]` Upgrade: Enhances your ship with a Triple-Shot spread.
  - `[H]` Health: Restores your lives (Max 8).
● **Cross-Platform:** Fully responsive with keyboard controls for PC and touch buttons for Mobile.
● **Hidden Easter Egg:** Win the game to unlock a special surprise and developer links! 🥚

## 🛠️ Tech Stack

➤ **HTML5:** Leveraging the `<canvas>` API for high-performance rendering.
➤ **CSS3:** Modern UI overlays, system font-stack optimization, and responsive design.
➤ **Vanilla JavaScript:** 60 FPS Game Loop (`requestAnimationFrame`), Object-Oriented Programming (ES6 Classes), and AABB Collision Detection.

## 🕹️ How to Play

1. Clone the repository or download the `index.html` file.
2. Open `index.html` directly in any modern web browser (Chrome, Edge, Safari, Brave).
3. **Desktop Controls:** Use `Left Arrow`/`Right Arrow` or `A`/`D` keys to move.
4. **Mobile Controls:** Use the on-screen ◀ and ▶ touch buttons.
